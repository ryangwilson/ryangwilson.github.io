Inline elements

Anchor tag <a>: This is a text link.

Bold-ness <b> and <strong>: The <strong> element is for content that is of greater importance, while the <b> element is used to draw attention to text without indicating that it's more important.

Italic-ness <i> and <em>: The <em> element represents stress emphasis of its contents, while the <i>element represents text that is set off from the normal prose, such a foreign word, fictional character thoughts, or when the text refers to the definition of a word instead of representing its semantic meaning.

Underline and Insert <u>, <ins>: The <u> u element is text with an unarticulated, though explicitly rendered, non-textual annotation. The The <ins> element represents a range of text that has been added to a document.

Strikethrough and Delete <s> and <del>: Use the <s> element to represent things that are no longer relevant or no longer accurate. The The <del> element should be used if the data has been deleted.

Inline Quotations and Citations<q> and <cite>: This text is a short inline quotation. This is an inline citation link to source

Highlights<mark>: The mark element indicates a highlight.

Variables<var>: The variable element, such as x = y.

Small text <small>: This small text is small for for fine print, etc.

Superscript <sup>: Superscript®.

Blockquotes

A block quotation (also known as a long quotation or extract) is a quotation in a written document, that is set off from the main text as a paragraph, or block of text.

It is typically distinguished visually using indentation and a different typeface or smaller size quotation. It may or may not include a citation, usually placed at the bottom.
